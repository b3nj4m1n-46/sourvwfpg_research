# Decks

Presentation source files for the SOURVWF research.

## Active proposal

**Workforce Pell at SOU** — proposal that Southern Oregon University adopt a Workforce Pell track under the brand "Applied Bioregional Learning." Audience: SOU administration + Rogue Valley community + (secondary asks to) HECC, the Workforce & Talent Development Board, and the Governor's Office.

Authors: Dr. Daniel DeFreez (SOU) · Haylee Ulrey (Autom Insights) · Benjamin Durham (White Rabbit).

Status: **Phase 0 / proposal sketch** — in stakeholder review.
